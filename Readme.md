## Запуск сервера
- Переместить директорию SamRLSim и сервера в одну директорию. Поменять путь до директории с кодом сервера в SamRLSim
- Устанавливаем [Python](https://www.python.org)
- Создаем виртуальное окружение ```python -m venv .venv```
- Устанавливаем библиотеки ```pip install -r requirements.txt```
- Запускаем ```py main.py```
